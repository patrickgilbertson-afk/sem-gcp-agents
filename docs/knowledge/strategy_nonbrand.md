# NonBrand Campaign Strategy

## Objectives

1. **Acquisition**: Reach users searching for solutions (not yet brand-aware)
2. **CPA Efficiency**: Maintain target CPA while scaling volume
3. **Quality Traffic**: Focus on high-intent keywords that convert

## Target Metrics

- **CPA**: $80-$120 (varies by geo and theme)
- **ROAS**: 400%+ (4:1 return on ad spend)
- **Conversion Rate**: 5-10% (lower than brand, higher than competitor)
- **Quality Score**: 7+ average (good relevance and experience)
- **CTR**: 3-5% (decent for non-brand terms)

## Keyword Strategy

### Keyword Themes

#### AI Code / AI Coding Assistant
**Intent**: Users seeking AI-powered coding tools
- "ai code completion"
- "ai coding assistant"
- "ai pair programmer"
- "ai code generator"
- "machine learning code assistant"

**Match Types**:
- Exact match: High-intent, high-converting
- Phrase match: Scaled reach with good control
- Broad match: Limited use, only with conversion-based smart bidding

#### Developer Tools
**Intent**: General developer productivity
- "code editor with ai"
- "intelligent ide"
- "code autocomplete tool"
- "developer productivity software"

#### Cloud IDE
**Intent**: Browser-based development solutions
- "cloud ide"
- "online code editor"
- "browser based ide"
- "collaborative coding platform"

### Negative Keywords (Universal)

**Irrelevant Intent**:
- "free" (only free-tier seekers, low conversion)
- "crack" / "pirate" / "torrent" (illegal, no conversion)
- "tutorial" / "how to" (informational, not transactional)
- "review" / "comparison" (research phase, low immediate conversion)
- "job" / "salary" / "career" (job seekers, not buyers)

**Wrong Product Category**:
- "no code" / "low code" (not our product)
- "wordpress" / "wix" / "shopify" (website builders, not code editors)
- "excel" / "spreadsheet" (different tool category)

**Poor Quality Traffic**:
- "cheap" / "discount" (price-focused, unlikely to convert to paid)
- "for kids" / "for beginners" (not target audience)

## Ad Copy Guidelines

### Headlines
**Primary**: Lead with benefit
- "Code 10x Faster with AI"
- "AI-Powered Code Completion"
- "Intelligent Coding Assistant"

**Secondary**: Feature highlight
- "Multi-Language Support | Real-Time Collaboration"
- "Works with VS Code, JetBrains, Any IDE"
- "Secure | SOC 2 Certified"

**Tertiary**: Offer/CTA
- "Start Free 14-Day Trial"
- "30% Off Pro Plans This Month"
- "Trusted by 500K+ Developers"

### Descriptions
- **Problem/Solution**: "Stop writing boilerplate. Let AI handle repetitive code."
- **Specificity**: "Support for Python, JavaScript, TypeScript, Go, Java & more"
- **Trust**: "SOC 2 certified. GDPR compliant. Enterprise-ready."
- **CTA**: "Try free for 14 days. No credit card required."

```
AI-Powered Code Completion
Stop Writing Boilerplate. Code 10x Faster with AI.

```

## Bidding Strategy

**Target CPA Bidding**:
- Set target CPA based on geo and theme
- US: $100 target CPA
- UK: $90 target CPA
- DE: $110 target CPA
- FR: $120 target CPA

**Learning Phase**:
- New campaigns need ~30 conversions before bidding optimizes
- During learning, may see higher CPA fluctuations
- Don't panic if CPA is $150-$200 in first 2 weeks

**Budget Pacing**:
- NonBrand gets 60% of total budget
- Distribute across themes based on performance:
  - AI Code: 50% (highest converting)
  - Developer Tools: 30%
  - Cloud IDE: 20%

## Quality Score Optimization

Target: **7+ average Quality Score**

### Expected CTR
- 3-5% is healthy for non-brand terms
- Below 2%: Ad copy not compelling or keyword too broad
- Above 6%: Excellent, may indicate very high intent keywords

### Ad Relevance
- Mirror keyword language in ad copy
- For "ai code completion", use "AI Code Completion" in headline
- For "coding assistant", use "Coding Assistant" in headline
- Use dynamic keyword insertion sparingly (can create awkward phrasing)

### Landing Page Experience
- Send AI-specific keywords to AI features page, not generic homepage
- Fast load times (<2 seconds)
- Mobile-optimized (50%+ of traffic is mobile)
- Clear CTA above the fold
- Social proof (reviews, logos, testimonials)

## Conversion Goal Alignment

**Primary Conversion**: SQC Org Creates (Self-Serve Qualified Customer - Organization)

**Why This Metric**:
- Indicates user created an organization/team (not just individual account)
- Higher LTV than individual free-tier users
- Strong signal of product-market fit

**Secondary Conversions** (tracked but not primary optimization target):
- Free trial starts
- Pro upgrades
- Contact sales form submissions

## Sync Group Considerations

NonBrand campaigns are **SYNCED** across geos:
- Same keywords across US, UK, DE, FR (translated where needed)
- Same ad copy themes (localized for language/culture)
- Coordinated expansion strategy

**Template Campaign**: Usually US market (most data, fastest learning)

**Geo Rollout**:
1. **US**: Launch and optimize first (2-4 weeks)
2. **UK**: Clone and launch (English, minimal changes)
3. **DE/FR**: Translate and launch (needs localization)
4. **Emerging markets**: Launch after proven success in Tier 1 geos

**Keyword Propagation**:
- When new high-performing keywords identified in US, add to all geos
- When poor performers identified, remove from all geos
- Sync happens weekly via automated process

## Expansion Strategy

### New Keyword Discovery

**Search Term Mining** (weekly):
- Review search term reports for all geos
- Identify converting search queries not yet targeted as keywords
- Add as phrase match or exact match keywords

**Thresholds**:
- Add if: ≥3 conversions, CPA ≤ 1.5x target
- Test if: 10+ clicks, CTR ≥ 3%, no conversions yet
- Negative if: 20+ clicks, 0 conversions, CPA n/a

### Ad Group Expansion

**When to create new ad group**:
- Distinct keyword theme emerges (e.g., "voice coding" becomes a trend)
- Existing ad groups have 15+ keywords (too broad, hard to optimize)
- Conversion rates vary significantly within ad group

**When NOT to expand**:
- Insufficient volume (don't create ad group for 1-2 keywords)
- Overlapping intent (consolidate, don't fragment)
- Too early (need at least 30 days of data)

## Red Flags / Escalation Triggers

Escalate to marketing lead if:
1. **CPA exceeds $180**: Unsustainable, pause and investigate
2. **Quality Score avg drops below 5**: Major relevance or LP issue
3. **Conversion rate drops below 3%**: Traffic quality degraded
4. **Daily spend increases >20% unexpectedly**: Potential click fraud or broad match runaway
